# Teacher Attendance & Digital CHR System

## 📌 About Project
AI-powered Teacher Attendance System using 
Face Recognition technology with Digital 
Class Hour Record (CHR) management.

## 🛠️ Technologies Used
- Python
- React JS
- OpenCV
- Face Recognition Library
- SQL Database

## ✨ Features
- Automated teacher attendance via face recognition
- Real-time face detection
- Digital CHR management
- In/Out time tracking
- Database management

## 📊 Results
- Face recognition accuracy: 94%+
- Real-time detection speed
